# Inventory Management System 

A desktop application built using Python, Tkinter and SQLite to manage product inventory. 

## Features 
- Add products
- View products
- Update products
- Delete products
- Search functionality
- Input validation

## Technologies Used 
- Python
- Tkinter
- SQLite

## How to Run
1. Download the project
2. Open the folder
3. Run:
   python main.py
